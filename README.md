# HyQBench
A benchmarking suite for Hybrid CV-DV quantum systems

Built using Bosonic Qiskit (https://github.com/C2QA/bosonic-qiskit) and QuTip

## Install

Use a Python virtual environment to install compatible version of QisKit.
